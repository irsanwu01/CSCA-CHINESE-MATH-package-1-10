## 🚀 Cara Penggunaan | How to Use

### Untuk Pengguna Baru | For New Users

#### 1️⃣ Registrasi Akun
- Buka aplikasi di `https://username.github.io/csca-exam`
- Klik **"Register"** / **"注册"**
- Isi:
  - Nama lengkap
  - Email aktif
  - Password (min. 6 karakter)
  - CSCA ID (opsional)

#### 2️⃣ Aktivasi Akun
- Setelah registrasi, status akun = **"pending"**
- Hubungi admin via WhatsApp untuk konfirmasi
- Admin akan **approve** akun Anda (1x24 jam)

#### 3️⃣ Login & Mulai Ujian
- Login dengan email dan password
- Pilih paket soal
- Klik **"Start Exam"**
- Kerjakan soal (timer berjalan)
- Lihat hasil setelah selesai
